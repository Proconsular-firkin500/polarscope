## 📋 About the application

Polarscope provides a way to see data from your RPLIDAR C1 sensor. It displays live measurements on your computer screen using a clear polar plot. The software shows scan results as they arrive. It includes a dark theme to reduce eye strain during use. You can save your scan data to a file for review later. The program also creates snapshots of the scan view as images.

## 💻 System requirements

Your computer needs to meet these basic standards to run the software:

* Operating System: Windows 10 or Windows 11.
* Processor: Intel Core i3 or equivalent.
* Memory: 4 gigabytes of RAM.
* Storage: 200 megabytes of free space.
* Hardware: One available USB port for your RPLIDAR C1.

## ⚙️ Setup and hardware

Connect your RPLIDAR C1 to your computer using a USB cable. Wait for Windows to identify the hardware. If Windows asks for a driver, visit the website of the sensor manufacturer to get the latest driver file. Do not move the sensor after you connect it to ensure the scan remains steady.

## 🚀 Running the software

1. Open the extracted folder from your download.
2. Double-click the file named polarscope.exe to start the program.
3. The main window appears on your screen.
4. Select the correct serial port from the dropdown menu in the settings bar.
5. Choose the correct baud rate for your C1 unit.
6. Click the Start button.
7. The plot window displays the laser scan data.

## 📊 Using features

The interface offers tools to manage your data stream. Use these options to control the display and record information.

### View controls
The central area shows the polar plot. It updates with every rotation of the sensor. Use your mouse scroll wheel to zoom in or out on the scan. Click and drag the plot to shift the view.

### CSV recording
You save scan points to a CSV file to open them in programs like Excel. Click the Record button to start saving data. The software saves the file in the application directory. Click Stop to finish the recording.

### PNG snapshots
You capture an image of the current scan display. Click the Snapshot button to trigger this action. The application saves the file as a PNG image in the same location as your program.

## 🔧 Troubleshooting

If the software fails to start, check the following items.

### Hardware connection
Ensure the USB cable connects to the sensor and the computer. If the sensor LED does not blink, check the cable for damage. Try a different USB port on your machine if the software does not detect a serial port.

### Permission issues
Some security settings block programs from running. If Windows shows a warning, click More Info and then select Run Anyway. This confirms you trust the application source.

### Missing files
If the software reports a missing file error, return to the project website and download the full set again. Ensure you extract every file from the compressed folder before you run the program.

## 🛡️ Privacy and data

Polarscope works locally on your computer. It does not send scan data to external servers or the internet. Your recorded files remain on your local hard drive. You control all data created during use.

This software supports the RPLIDAR C1. It might not function with other models or brands.

Where does the software save my files?
It saves recordings in the same folder where you placed the application files.

How do I close the program?
Click the X button in the top corner of the window to shut down the application safely.

Does the software need an internet connection?
No. All features work while you remain offline.
